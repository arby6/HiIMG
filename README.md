## hiimg
*(hide inside images)*

hiimg is a small steganography tool that lets you hide text inside images and recover it later.

it works by embedding your message directly into image data, so the result still looks like a normal image but it secretly carries your text inside.

---

## what it does

* load png or jpg images
* type a hidden message
* inject text into the image
* extract hidden text back anytime using hiimg

---

## features

* drag & drop image support
* live image preview
* text injection & extraction
* one-click copy for extracted text
* capacity indicator (how much text fits)

---

## notes

* png is strongly recommended (best results)
* avoid compressing images after embedding data
* larger images = more space for hidden text
* editing the image after injection may destroy hidden data

---

## how to run

```bash
install requirements
run main.py
```

that’s it.

---

## how it feels

drop an image in → hide a message → it just looks like a normal picture
but it’s not.
